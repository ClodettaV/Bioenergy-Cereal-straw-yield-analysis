The objective of this project was to collect open-file data related to biomass material in Western Australia that can be used for energy production, discover interesting data trends. A machine learning model has been used to obtain a prediction on how much yield can be produced in a specific area, given a certain amount of rain and a given soil pH.

Specifically, the study focussed on cereal crops in WA. Unfortunately, the dataset for straw yield is limited to 25 datapoints. The data obtained for rainfall and soil pH are more extensive, with more data points within each shires and data from more shires are also available. To be able to merge the datasets, rainfall and soil pH need to be averaged per shire, losing therefore more detailed information across the shires.

The geographical distribution of the rainfall dataset and the straw yield dataset shows that rain tends to be more intense inland and the straw yield more abundant toward the coast.
Soil ph seems to be quite scattered across the area. It would be interesting to plot all sample points for soil pH available from the dataset in a future analysis.

The machine learning model chosen use a multi-linear regression algorithm. The limited number of data results in a less accurate machine learning model prediction. However it is possible to identify a negative relationship between rainfall and yield with the data analysed.
Soil pH vs rainfall and Soil pH vs straw yield cross-correlation plots show a more scattered distribution, with no strong relationship between the data.
In the future, it would be beneficial to test the data with a different model.
The application build in flask uses the machine learning model to give a prediction of the yield/ha given a certain value of rain and soil pH. Javascript has been used for the visualisation. Limitations - The number of straw yield was not very extensive. - Plot all data for soil pH and rainfall to discover potential trends - Use groundwater salinity data as a new input to the ML model. references
