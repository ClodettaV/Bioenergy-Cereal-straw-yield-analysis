Aim of the project and process


The aim of this project is to evaluate the biomass available in WA and to analyse factors that contribute to growth of cereals in this instance. This includes simple analysis for now of cereal straw yield per hectares in WA crops, given the limited amount of data found. I would like to answer the following questions:

- Where do we find the best yield in WA?
- Soil pH has an impact on crops. How this is affecting the crops in WA?
- Has different amount of rainfall an effect on the soil and therefore on the amount of crops that grow in WA?

To answer these questions a combination of datasets have been sourced from a CSV database (loaded in AWS in S3 bucket) and an API endpoints. After cleaning and merging data, the resulting data-frames have imported into jupyter notebook and input into a linear regression model using scikit-learn. The prediction obtained from the created model has been used to build an application with javascritp and visualize the results on a html page. A flask application has been used to return prediction data to the users via REST API end point. The data have also been used in Tableau to analyze each dataset and to discover relationships between the data through plots and maps.
