Objectives and solution architecture


The aim of this project is to evaluate the straw yield potential for WA through the analysis of factors that contribute to the production of cereals, such as rainfall and soil pH. This analysis will aim to answer the following questions:

- Where do we find the highest yield in WA?
- It is known that soil pH has an impact on plants growth. How is this affecting the yield in WA?
- How do rainfall influence soil pH and straw yield in WA?

A combination of three datasets, sourced from the public domain, have been merged. These include CSV files and data from API endpoints. After cleaning and merging data, the resulting data-frames have imported into jupyter notebook and input into a machine learning model using scikit-learn Python library. The prediction obtained from the model has been used to build a tool also using javascritp to help visualize the results on this website. A flask application has been used to return prediction data to the users. Tableau has been used to analyze each dataset and to discover relationships between the data, using plots and maps.
